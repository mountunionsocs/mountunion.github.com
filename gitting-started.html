Welcome to the SOCS Gitting Started Guide. We're going to help you set up a GitHub account and join our SOCS GitHub organization. That way you can download any of our source code or organization documents, make changes, and push them back to the master branch. Teamwork for the win!

Wait, what's GitHub?

GitHub is a social code sharing site. You make an account, upload some of your open source projects, and then others can contribute, fix bugs and do all sorts of fun things.

What's even better is that Netbeans recently started supporting git (more on that in a bit), which means that all our code on GitHub can easily be downloaded, edited and pushed back all from our lab PCs.
